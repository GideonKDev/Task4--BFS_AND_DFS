# My Project    
This project is a simple python script to perform a Depth First Search (DFS) on a graph. The graph is represented as an adjacency list, and the DFS algorithm is implemented using recursion. It also has a Breadth First Search (BFS) implementation using a queue. The project is designed to demonstrate the basic concepts of graph traversal algorithms.

## Features
- Depth First Search (DFS) implementation
- Breadth First Search (BFS) implementation
- Graph representation using an adjacency list

## Usage
Simply run the scripts for both BFS and DFS.
